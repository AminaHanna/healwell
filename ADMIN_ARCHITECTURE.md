## Database Operations

### Blog Operations

#### Create Blog
```javascript
const { error } = await supabase
  .from('blogs')
  .insert([blogData]);
```

#### Read Blogs
```javascript
const { data, error } = await supabase
  .from('blogs')
  .select('*')
  .order('created_at', { ascending: false });
```

#### Update Blog
```javascript
const { error } = await supabase
  .from('blogs')
  .update(blogData)
  .eq('id', blogId);
```

#### Delete Blog
```javascript
const { error } = await supabase
  .from('blogs')
  .delete()
  .eq('id', blogId);
```
